# Skills

Pre-defined skill patterns for Agent HQ development.

## Data Schema Skills

### Vendor Management
- Add new vendors to `lib/catalog.ts`
- Update vendor information
- Link vendors to systems

### System Management
- Add new agent systems with versions
- Update system capabilities
- Mark systems as deprecated

### Template Management
- Add new agent templates
- Update template manifests
- Manage template tags

## UI/Component Skills

### Landing Page
- Modify hero section
- Add new featured sections
- Update stats counters

### Card Components
- Create new card types
- Add hover interactions
- Style with emerald accent

## DevOps Skills

### GitHub Pages Deployment
- Configure basePath
- Set up custom domains
- Debug build failures

### Build Optimization
- Reduce bundle size
- Optimize images
- Add new static routes
